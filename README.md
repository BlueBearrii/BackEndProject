## Routing

| Route name       | Method   | Path      | Description                                   |
| ---------------- | -------- | --------- | --------------------------------------------- |
| **Register**     | **POST** | /register | For register new user firebase authentication |
| **Login**        | **POST** | /login    | For login with Firebase                       |
| **Get all user** | **GET**  | /users    | Get all user data from mySql                  |
